# 🏗️ E-Tendering System (Easy Tendering Web)

## 📌 Project Overview
The **E-Tendering System** is a web-based application developed for academic course work.  
It digitizes the traditional tendering process, allowing vendors, evaluators, and admins to interact in a transparent and efficient way.

This system supports tender creation, bidding, evaluation, and management in a centralized platform.

---

## 🚀 Features

### 👤 User Roles
- 🧑 Admin
- 🏢 Vendor
- 🧑‍⚖️ Evaluator

### 🛠️ Core Features
- User registration & login system
- Tender creation & management
- Bid submission system for vendors
- Bid evaluation system
- Admin dashboard
- Messaging system between users
- Payment module (demo/academic)
- Tender status tracking
- Data management using MySQL
